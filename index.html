<h1>Repl.it is now Replit.com!</h1>
When we started Replit, it was just a website where you could run code online. So the name, "repl.it", made a lot of sense. Today, you can do so much more than run code. You can teach entire univerities, share your projects and help other people solve programming problems. We've seen our website utilized in ways we'd never imagined. Now we want our domain, the first thing people see, to reflect the progress we've made. We're moving repl.it to replit.com. Don't worry, this change will have zero impact on you. You'll stay logged in, and we'll redirect all repl.it connections to replit.com. For more information about the change, checkout our blog post.

    In 1991, HTTP/.9 was introduced to the world. It wouldn=E2=80=99t be until 1994 when HTTPS would be invented to encrypt online traffic.
